# Contributing

There are a lot of different ways to contribute to Automator project.

## Make changes to Automator.NET code or docs

Fork the project, make a change, and send a pull request!

## Submit bug reports or feature requests

Just use the GitHub issue tracker to submit your bug reports and feature requests.
If you are submitting a bug report, please follow the [issue template](https://github.com/dtopuzov/automator.net/issues/new).
